Detailed Description

Proxy group to use in place of Fl_Group in the interactive window.

In an interactive environment, groups should not automatically resize their children. This proxy disables the layout of children by default. Children layout propagation may be enable temporarily by incrementing allow_layout before resizing and decrementing it again afterwards.
