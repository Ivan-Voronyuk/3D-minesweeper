static void Fl_Mac_App_Menu::custom_application_menu_items ( const Fl_Menu_Item *  m)
static

Adds custom menu item(s) to the application menu of the system menu bar. They are positioned after the "Print Front Window / Toggle printing of titlebar" items, or at their place if they were removed with Fl_Mac_App_Menu::print = "".

Parameters
mzero-ending array of Fl_Menu_Item 's.
