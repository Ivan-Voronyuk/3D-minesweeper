◆ override_visible_

uchar Fl_Widget_Type::override_visible_
protected

This variable is set for visible windows in batch mode. We can't open a window in batch mode, even if we want the "visible" flags set, so we need a second place to store this information while also disabling the output of the "hide" property by the Widget Type.
