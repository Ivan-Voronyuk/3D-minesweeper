◆ read_word()

const char * Fd_Project_Reader::read_word ( int  wantbrace = 0)

Return a word read from the .fl file, or NULL at the EOF.

This will skip all comments (# to end of line), and evaluate all \xxx sequences and use \ at the end of line to remove the newline.

A word is any one of:

  • a continuous string of non-space chars except { and } and #
  • everything between matching {...} (unless wantbrace != 0)
  • the characters '{' and '}'
Parameters
[in]wantbraceif set, reading a { as the first non-space character will return the string "{", if clear, a { is seen as the start of a word
Returns
a pointer to the internal buffer, containing a copy of the word. Don't free the buffer! Note that most (all?) other file operations will overwrite this buffer. If wantbrace is not set, but we read a leading '{', the returned string will be stripped of its leading and trailing braces.
