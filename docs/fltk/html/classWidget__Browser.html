Detailed Description

A widget that displays the nodes in the widget tree.

The Widget Browser is derived from the FLTK basic browser, extending tree browsing functionality by using the depth component of the double linked list of Fl_Type items.

◆ handle()

int Widget_Browser::handle ( int  e)

Override the event handling for this browser.

The vertical mouse position corresponds to an entry in the type tree. The horizontal position has the following hot zones:

  • 0-3 is the widget frame and ignored
  • the next hot zone starts 12*indent pixels further to the right
  • the next 13 pixels refer to the arrow that indicates children for the item
  • 18 pixels follow for the icon
  • the remaining part is filled with text
Parameters
[in]ethe incoming event type
Returns
0 if the event is not supported, and 1 if the event was "used up"
