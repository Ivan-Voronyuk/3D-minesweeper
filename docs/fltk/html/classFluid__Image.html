void Fluid_Image::write_static ( Fd_Code_Writer &  f,
int  compressed 
)

Write the static image data into the soutrce file.

If compressed is set, write the original image format, which requires linking the matching image reader at runtime, or if we want to store the raw uncompressed pixels, which makes images fast, needs no reader, but takes a lot of memory (current default for PNG)

Parameters
compressedwrite data in the original compressed file format
