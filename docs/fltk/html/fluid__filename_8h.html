Detailed Description

File names and URI utility functions for FLUID only.

This file declares all fl_filename* functions using Fl_String and also includes the main header file <FL/filename.H>.

Note
This file contains some filename functions using Fl_String which which are used in FLTK 1.4.x but will be removed in the next minor or major release after 1.4.x (i.e. 1.5 or maybe 4.0).
This entire file should become obsolete in 1.5 or higher, whatever the next release after 1.4.x will be. We'll use std::string instead!
